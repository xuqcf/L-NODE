# L-NODE

> A public Obsidian vault of my X-NODE version — my living knowledge base, built in the open.

L-NODE is where I document everything I learn as a self-taught developer, CS student, and builder.  These aren't polished tutorials or blog posts but my actual working notes whenever I'm learning something, the kind I'd want to have found when I was starting out. If something in here helps you, that's the whole point.

The vault covers programming, AI/ML, Linux, STEM, technology, personal growth, philosophy, and whatever else I'm currently obsessing over. It grows as I do.

If you find it useful, consider starring the repo, it helps more people find it. ⭐
